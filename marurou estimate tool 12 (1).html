MODE A · VENDOR IMPORT

業者見積から作成 // IMPORT

業者からもらったPDF見積（またはExcel）を取り込み、係数（×掛け）を指定すると自社の見積価格が一発計算されます。最終的に行ごとに微調整して仕上げてください。

STEP 1 · 業者見積を取り込む PDFまたはExcel/CSVを選択 or ドラッグ

業者見積からの自動取込は補助機能です。フォーマットによっては自動抽出できない場合があります。
うまくいかない場合は「＋ 手入力で作成」から始めて、業者見積を見ながら明細を1行ずつ追加してください。

📥 クリックでファイルを選択 or ここにドラッグ&ドロップ
対応: PDF (.pdf) / Excel (.xlsx, .xls) / CSV (.csv)
取込せずに、最初から手入力で作成することもできます
※明細はクリアされます

STEP 2 · 案件情報 出力時に反映

※ 宛名には敬称を含めず、会社名・お名前のみを入力してください。敬称は右の欄で選びます（プレビューでは「宛名 + 敬称」が表示されます）。
テンプレートを選んで「反映」を押すと備考欄に挿入されます。挿入後は自由に編集できます。

MODE B · PRICE BOOK

単価表から作成 // BUILD

被害部屋を選んでセクションを作り、各セクションに単価表から品目を追加していきます。

STEP 1 · 被害部屋を選択 複数選択可。後から追加・削除もできます

選択した部屋ごとにセクションが作られます。「表紙」「内訳明細」「仮設・荷物移動・養生費」「一般管理費」は自動で追加されます。
※明細はクリアされます

MODE C · INVOICE

請求書発行 // INVOICE

見積データ（JSON）を取り込んで請求書を作成します。複数の見積をまとめて1枚の請求書にすることもできます。

STEP 1 · 見積データを取り込む 「💾 一時保存」で保存したJSONを選択

タブ①②で保存した見積JSONファイルを取り込みます。複数取り込むと1枚の請求書にまとまります。

📥 クリックでJSONを選択 or ここにドラッグ&ドロップ
対応: 見積JSON (.json) ／ 複数選択・複数回追加できます

取り込んだ見積

STEP 2 · 請求書の設定

小計（税抜）¥0
消費税¥0
請求金額（税込）¥0
クリックでセル編集 / PDF化時は印刷設定で余白「なし」・背景のグラフィックにチェック

📄 PDF化の前に

これからブラウザの印刷ダイアログが開きます。
プレビューと同じ見た目のPDFにするため、印刷ダイアログで以下を確認してください：

1. 「送信先」を 「PDFに保存」 に
2. 「ページ」を 「すべて」 に
3. 「余白」を 「なし」 に
4. 「オプション」の 「背景のグラフィック」にチェック を入れる

※ これらの設定はブラウザが記憶するため、初回のみ設定すれば次回以降は自動で適用されます。

💾 PDFと同時に、編集用のJSONファイルも自動ダウンロードされます。
（あとで請求書発行や再編集に使えるよう、PDFとセットで保管してください）
